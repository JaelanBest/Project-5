How to run:
1. Download and unzip the folder
2. Open index.html using Google Chrome (recommended)

How to test mobile and desktop:
- Open the site in Chrome
- Right click anywhere and click Inspect
- Click the phone icon (Toggle Device Toolbar) at the top left
- Set width to around 375px for mobile view
- Set width to around 1280px for desktop view
- Refresh the page after switching sizes if needed

What is included:
- Responsive mobile and desktop layout
- Working navigation and hamburger menu
- Real time search with suggestions
- Loading state when posting
- Smooth animations (hover and fade in)
- Dark mode toggle
- Data saves using localStorage

Notes:
- Search updates as you type and clears when empty
- Suggestions appear under the search bar
- Mobile layout stacks properly and desktop uses sidebar
- Mobile view was tested using Chrome DevTools at 375px width
- Desktop view was tested at 1280px width
