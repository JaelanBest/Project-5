# Queen City List - Final Functional Version

## Updates in this version
- Category page counts are now data-driven and every dropdown category has at least 3 listings.
- Each major category box totals at least 15 listings through its dropdown categories.
- Listings include visible owner contact information plus Contact seller, Email, and Call actions when available.
- Contact seller opens a simple modal that can launch the buyer's email app with a prefilled message.
- Listings page includes a real clickable Leaflet map.
- Clicking a numbered map marker filters listings by location.
- Show all areas resets the map filter.
- Dark mode remains symbol-only and persists between pages.
- Layout uses responsive, zoom-safe styles for accessibility.

## How to run
1. Unzip the folder.
2. Open `index.html` in Google Chrome.
3. Visit `categories.html` to see category counts.
4. Visit `Listings.html` to use the interactive map and contact sellers.

## How to test
- Click a category dropdown item to view listings in that category.
- Open the Listings page and click a map marker such as Charlotte or Concord.
- Click Show all areas to reset the filter.
- Click Contact seller on any listing.
- Test at 200% zoom to confirm cards stack and remain readable.
